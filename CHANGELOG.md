# Changelog

## [0.1.0] - 2026-02-15

### Added
- Release packaging pipeline for Darija TTS exports.
- `scripts/pack_release.py` to build dataset and open-source release bundles.
- `scripts/validate_dataset.py` for structural and audio-format validation.
- `scripts/make_report.py` for dataset metrics, plots, and markdown reporting.
- Open-source repo assets: `LICENSE`, `.gitignore`, `requirements.txt`.
- Documentation bundle: `README.md`, `docs/report.md`, `examples/sample_usage.md`,
  and `colab/colab_steps.md`.
- Optional CI workflow: `.github/workflows/validate.yml`.
